# 计费管理系统 (C语言版)

使用纯C99实现的网吧计费管理系统，采用二进制文件读写替代SQLite数据库。

## 快速开始

### 编译
```bash
compile.bat
```
或
```bash
gcc -std=c99 -Wall -Wextra -Iinclude -finput-charset=UTF-8 -fexec-charset=GBK src\*.c -o app.exe -lws2_32
```

### 运行
```bash
app.exe
```

### 默认管理员账号
- 用户名: `admin`
- 密码: `123456`

## 功能模块

| 模块 | 功能说明 |
|------|----------|
| 管理员管理 | 登录、查看管理员列表、添加/删除管理员、修改密码 |
| 卡务管理(链表) | 注册新卡、查询卡信息、列出所有卡、注销卡 |
| 卡务管理(文件) | 查询账户、账户信息变更、注册账户、删除账户、账户列表 |
| 上下机服务 | 用户上机、下机、自动计费 |
| 充值退费 | 充值、退费操作 |
| 查询业务 | 查询所有卡、上下机记录、营业额记录、用户全记录、详细记录 |
| 计费标准设置 | 查看套餐列表、新增套餐、删除套餐、设置默认套餐 |

### 查询业务详情

1. **查询所有卡** - 显示所有有效卡的信息
2. **查询上下机记录** - 显示所有上下机记录，日志保存至 `uselogYYYY.log`
3. **查询营业额记录** - 显示所有财务交易记录（含充值/退费/消费），日志保存至 `salesYYYY.log`
4. **用户全记录** - 显示每个用户的汇总信息（上机次数、充值次数、消费次数）
5. **详细记录** - 按卡号查询该用户的完整记录（账户信息、上下机明细、财务明细）

## 技术栈

- **语言**: C99
- **编译器**: GCC (MinGW)
- **存储**: 二进制文件 (`.bin`) + 文本文件 (`.txt`)
- **数据结构**: 链表

## 数据文件

| 文件 | 说明 |
|------|------|
| `data/accounts.bin` | 账户信息 |
| `data/finance.bin` | 财务记录 |
| `data/loginout.bin` | 上下机记录 |
| `data/billing.txt` | 计费套餐配置 |
| `data/admins.bin` | 管理员信息 |
| `uselogYYYY.log` | 上下机查询日志（YYYY为年份） |
| `salesYYYY.log` | 营业额查询日志（YYYY为年份） |

## 与原C++版本的区别

- ❌ 移除SQLite依赖，改用二进制文件读写
- ❌ 移除`std::vector`，改用手动实现的链表
- ❌ 移除`std::string`，改用`char[]`固定数组
- ✅ 纯C99标准，无C++特性
- ✅ 无额外依赖库（仅使用GCC标准库）

## 计费套餐

默认套餐采用分段计费：
- 0-1小时: 1.8元/小时
- 1-3小时: 1.2元/小时
- 3小时以上: 0.6元/小时

支持自定义多套套餐，按实际使用分钟数计算费用。
