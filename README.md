# 计费管理系统 (C语言版)

使用纯C99实现的计费管理系统，采用二进制文件读写替代SQLite数据库。

## 快速开始

### 编译
```bash
compile.bat
```

### 运行
```bash
app.exe
```

### 默认管理员账号
- 用户名: `admin`
- 密码: `123456`

## 功能模块

| 模块 | 功能 |
|------|------|
| 管理员管理 | 登录、改密码、添加/删除管理员 |
| 账户服务 | 查询、注册、注销、修改账户 |
| 上下机服务 | 用户上机、下机、自动计费 |
| 计费标准管理 | 新增/查询/修改/删除计费套餐 |
| 财务服务 | 充值、退费、消费记录、营业额统计 |
| 查询统计 | 营业额日志、上下机日志、按日期查询 |

## 技术栈

- **语言**: C99
- **编译器**: GCC (MinGW)
- **存储**: 二进制文件 (`.bin`)
- **数据结构**: 链表

## 项目结构

```
├── include/          # 头文件
├── src/              # 源文件
│   └── services/     # 服务层实现
├── data/             # 数据存储 (运行时自动创建)
├── compile.bat       # 编译脚本
└── app.exe           # 可执行文件
```

## 与C++版本的区别

- ❌ 移除SQLite依赖，改用二进制文件读写
- ❌ 移除`std::vector`，改用手动实现链表
- ❌ 移除`std::string`，改用`char[]`固定数组
- ✅ 纯C99标准，无C++特性
- ✅ 文件按魔数+版本标识，支持数据完整性校验
